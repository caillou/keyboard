  Detect-and-handle that case cleanly (e.g. warn, or replace only when it's a
  symlink or absent). IMPORTANT: do NOT "fix" it with a blind `rm -rf` of a real
  directory.

  presence only (unzip succeeds). Optionally strengthen to record/verify the
  upstream commit (it already fetches it) so a silently-wrong fetch is
